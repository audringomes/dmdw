XSLT Stylesheets

 Scroll Prev Top Next More

DMDW contains the following xsl version 2.0 stylesheets to to transform the dmdw xml into html.

The files can be found in dm_doc_writer/resources/base/xsl:

 

xsl-file

Purpose

Remark

Fileset1.xsl

Main template that creates multiple html pages

Version 1 only

Fileset2.xsl

Main template that creates multiple html pages.

See FileSet2.xsl.

Version 2 (current)

Fileset3.xsl

Main template that creates multiple html pages, but without dd_version prefix and 'error page'.

When


ChapterTemplates.xsl

The named templates for every kind of page, for example "CollectionPages"

ProcessingTemplates.xsl

All templates for processing the DMDW xml and svg data.

index.xsl

Creates the 'index page' when multiple DMDW xml files are processed.

Common.xsl

Common named templates for:

- buttons

- head section of html pages

- top section of pages

- footer section of pages

Custom.xsl

Contains custom named templates for extra information on the type of pages

Loaded after common

TopologyFunctions.xsl

Special topology functions

Version 2

SingleFile.xsl

Transforms XML into one html page

Obsolete
